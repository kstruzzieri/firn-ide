Implementation Notes

  • Add .ide--accent-{color} class to the root element to swap the entire theme
  • Accent cascades through: header tint, tab indicators, selections, scrollbars, buttons, file tree highlights
  • The Flux logo gradient stays consistent (brand identity) — only the workspace accent changes
  • Deep Ocean base theme remains constant across all workspaces
  • Accent colors: blue, green, cyan, orange, purple, amber
